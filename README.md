# Mirror Moon - 镜之月

AI驱动的无限流恐怖乙女游戏

## 项目结构

```
mirror-moon/
├── index.html          # 主页面
├── package.json        # 项目配置
├── css/
│   └── main.css       # 主样式文件
├── js/
│   └── main.js        # 主逻辑文件
└── assets/
    ├── images/        # 图片资源
    └── sounds/        # 音效资源
```

## 功能特性

### 已实现基础功能
- ✅ 主菜单界面
- ✅ 游戏状态管理(生命值/理智/能量)
- ✅ 聊天对话系统
- ✅ 地图系统(带战争迷雾)
- ✅ 屏幕方向适配(竖屏/横屏)
- ✅ 数据本地存储

### 计划功能
- ⏳ Live2D角色集成
- ⏳ 360°环境探索
- ⏳ AI对话系统(接入大语言模型)
- ⏳ 动态副本生成
- ⏳ 多NPC交互
- ⏳ 动画UI效果

## 快速开始

1. 安装依赖(可选,仅用于本地开发服务器)
```bash
npm install
```

2. 启动开发服务器
```bash
npm start
# 或
npm run dev
```

3. 直接打开 `index.html` 也可以运行

## 游戏控制

- 输入框: 输入对话或指令
- 回车/发送按钮: 发送消息
- 常用指令:
  - "你好" - 与AI队友打招呼
  - "探索" - 探索环境
  - "地图" - 打开地图
  - "状态" - 查看当前状态

## 技术栈

- HTML5 + CSS3 + JavaScript (纯原生)
- Canvas API (地图绘制)
- LocalStorage (数据持久化)
- Live2D Cubism SDK (计划)
- Three.js (计划,用于360°环境)

## 开发说明

### 屏幕方向
- 竖屏模式: 主界面,角色展示,聊天交互
- 横屏模式: 地图探索,副本解谜

### 状态系统
- 生命值(HP): 受伤时减少,归零游戏结束
- 理智值: 遇到恐怖事件减少,归零触发精神崩溃
- 能量值: 行动消耗,休息恢复
